# Screen Classification

This audit covers every original Stitch `code.html` screen found in the project root.

## Classification Key

- `PUBLIC_LAUNCH`: launch-ready public page.
- `PUBLIC_LAUNCH_WITH_COPY_FIXES`: useful public page, but only after removing booking-platform or prototype language.
- `FUTURE_PROTOTYPE_ARCHIVE`: non-launch concept or product prototype.
- `INTERNAL_ONLY_ARCHIVE`: admin, staff, analytics, moderation, or operations screen.
- `DUPLICATE_SOURCE_ONLY`: useful as source reference only, not a distinct launch page.
